# Microsoft Foundry 与 Azure 合作

**时间：** 2025年11月  
**主题：** Microsoft Foundry、Claude 接入 M365 Copilot 和 GitHub Copilot、300亿美元 Azure 算力

---

## 概述

2025年11月，Anthropic 与 **Microsoft** 宣布全面合作。**Claude** 被集成到 **Microsoft Foundry**、**Microsoft 365 Copilot** 和 **GitHub Copilot**。Microsoft 承诺 **300亿美元** **Azure 算力**并投资 **50亿美元** 于 Anthropic。

---

## 合作要素

| 组件 | 详情 |
|-----------|--------|
| **Microsoft 投资** | 50亿美元 |
| **Azure 算力承诺** | 300亿美元 |
| **Microsoft Foundry** | 企业 AI 平台 |
| **M365 Copilot** | Claude 接入 Office 生产力应用 |
| **GitHub Copilot** | Claude 用于代码生成 |

---

## Microsoft Foundry

**Microsoft Foundry** 是 Microsoft 的企业 AI 平台，提供：

- 包括 Claude 在内的前沿模型访问
- 定制模型部署和微调
- 企业安全与治理
- 与 Microsoft 365 和 Azure 的集成

---

## Claude 集成

### Microsoft 365 Copilot

- Claude 可在 Office 应用（Word、Excel、PowerPoint、Outlook 等）中使用
- 生产力辅助、摘要、起草
- 与 OpenAI 模型并行的多模型策略

### GitHub Copilot

- Claude 用于代码生成和辅助
- 补充或增强现有 Copilot 能力
- 开发者生产力 focus

---

## Azure 算力

**300亿美元 Azure 算力承诺**确保：

- 长期获得 Microsoft 云基础设施
- 大规模训练和推理能力
- Anthropic 的多云策略（AWS、Google、Azure）

# Microsoft Foundry and Azure Partnership

**Date:** November 2025  
**Topic:** Microsoft Foundry, Claude in M365 Copilot and GitHub Copilot, $30B Azure compute

---

## Overview

In **November 2025**, Anthropic and **Microsoft** announced a comprehensive partnership. **Claude** was integrated into **Microsoft Foundry**, **Microsoft 365 Copilot**, and **GitHub Copilot**. Microsoft committed **$30 billion** in **Azure compute** and invested **$5 billion** in Anthropic.

---

## Partnership Elements

| Component | Detail |
|-----------|--------|
| **Microsoft investment** | $5 billion |
| **Azure compute commitment** | $30 billion |
| **Microsoft Foundry** | Enterprise AI platform |
| **M365 Copilot** | Claude in Office productivity apps |
| **GitHub Copilot** | Claude for code generation |

---

## Microsoft Foundry

**Microsoft Foundry** is Microsoft's enterprise AI platform, providing:

- Access to frontier models including Claude
- Custom model deployment and fine-tuning
- Enterprise security and governance
- Integration with Microsoft 365 and Azure

---

## Claude Integrations

### Microsoft 365 Copilot

- Claude available within Office applications (Word, Excel, PowerPoint, Outlook, etc.)
- Productivity assistance, summarization, drafting
- Multi-model strategy alongside OpenAI models

### GitHub Copilot

- Claude used for code generation and assistance
- Complements or augments existing Copilot capabilities
- Developer productivity focus

---

## Azure Compute

The **$30 billion Azure compute commitment** ensured:

- Long-term access to Microsoft's cloud infrastructure
- Capacity for training and inference at scale
- Anthropic's multi-cloud strategy (AWS, Google, Azure)
