# Google Cloud 合作

**时间：** 2023年2月  
**主题：** Anthropic 与 Google Cloud 作为主要提供商的合作

---

## 概述

2023年2月，Anthropic 在 Google **3亿美元投资**后确立 **Google Cloud** 为其主要云基础设施提供商。该合作为训练和推理提供 **GPU** 和 **TPU** 访问。

---

## 合作要素

| 组件 | 详情 |
|-----------|--------|
| **提供商** | Google Cloud |
| **算力** | GPU、TPU |
| **投资** | Google 3亿美元（参见 [20230200-google-investment-300m.md](20230200-google-investment-300m.md)） |
| **股份** | 约 10% 给 Google |

---

## 技术能力

### GPU

- 访问 Google Cloud 上的 NVIDIA GPU 及其他 GPU 实例
- 用于模型训练和推理

### TPU（张量处理单元）

- Google 定制 AI 加速器
- 针对大规模机器学习工作负载优化
- 用于训练和 serving Claude 模型

---

## Vertex AI 集成

Claude 后来在 **Google Vertex AI** 上可用（参见 [20240300-claude-3-on-vertex-ai.md](20240300-claude-3-on-vertex-ai.md)），即 Google 的托管 ML 平台，提供无服务器 API 和企业部署选项。

---

## 多云演进

Anthropic 随后增加 **AWS**（2023年9月）和 **Microsoft Azure**（2025年11月）作为合作伙伴。Google Cloud 与 AWS 和 Azure 一起仍是关键提供商。

# Google Cloud Partnership

**Date:** February 2023  
**Topic:** Anthropic's partnership with Google Cloud as primary provider

---

## Overview

In **February 2023**, Anthropic established **Google Cloud** as its primary cloud infrastructure provider, following Google's **$300 million investment**. The partnership provided access to **GPUs** and **TPUs** for training and inference.

---

## Partnership Elements

| Component | Detail |
|-----------|--------|
| **Provider** | Google Cloud |
| **Compute** | GPUs, TPUs |
| **Investment** | $300M from Google (see [20230200-google-investment-300m.md](20230200-google-investment-300m.md)) |
| **Stake** | ~10% to Google |

---

## Technical Capabilities

### GPUs

- Access to NVIDIA GPUs and other GPU instances on Google Cloud
- Used for model training and inference

### TPUs (Tensor Processing Units)

- Google's custom AI accelerators
- Optimized for large-scale machine learning workloads
- Used for training and serving Claude models

---

## Vertex AI Integration

Claude was later made available on **Google Vertex AI** (see [20240300-claude-3-on-vertex-ai.md](20240300-claude-3-on-vertex-ai.md)), Google's managed ML platform, providing serverless APIs and enterprise deployment options.

---

## Multi-Cloud Evolution

Anthropic subsequently added **AWS** (September 2023) and **Microsoft Azure** (November 2025) as partners. Google Cloud remained a key provider alongside AWS and Azure.
