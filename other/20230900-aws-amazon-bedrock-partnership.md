# AWS 与 Amazon Bedrock 合作

**时间：** 2023年9月  
**主题：** Amazon/AWS Bedrock 战略合作

---

## 概述

2023年9月，Anthropic 宣布与 **Amazon Web Services (AWS)** 和 **Amazon Bedrock** 建立**战略合作**。该合作紧随 Amazon 最高 **40亿美元** 的投资。Claude 成为 **Bedrock** 上的旗舰模型，合作已服务 **1万+ 客户**。

---

## 合作要素

| 组件 | 详情 |
|-----------|--------|
| **平台** | Amazon Bedrock |
| **投资** | Amazon 最高 40亿美元 |
| **云** | AWS 为主要提供商 |
| **客户** | 1万+（截至合作里程碑） |

---

## Amazon Bedrock

**Amazon Bedrock** 是 AWS 的基础模型全托管服务。它提供：

- 通过 API 访问 Claude 及其他基础模型
- 无服务器推理
- 微调和定制
- 企业安全与合规

---

## 战略合作

- **Claude 为旗舰** – Claude 被定位为 Bedrock 上的领先模型
- **Trainium 和 Inferentia** – 使用 Amazon 定制 AI 芯片进行训练和推理
- **企业分发** – AWS 庞大的企业客户基础
- **全球可用性** – Bedrock 在 AWS 区域广泛可用

---

## 客户增长

该合作报告 **1万+ 客户**通过 Bedrock 使用 Claude，包括金融、医疗、科技等行业的企业。

---

## 后续发展

- **2024年11月：** Amazon 承诺额外 40亿美元（总计 80亿美元）——参见 [20241100-amazon-additional-4b.md](20241100-amazon-additional-4b.md)
- Claude 继续成为 Bedrock 上的核心产品

# AWS and Amazon Bedrock Partnership

**Date:** September 2023  
**Topic:** Amazon/AWS Bedrock strategic collaboration

---

## Overview

In **September 2023**, Anthropic announced a **strategic collaboration** with **Amazon Web Services (AWS)** and **Amazon Bedrock**. The partnership followed Amazon's investment of up to **$4 billion**. Claude became a flagship model on **Bedrock**, and the collaboration grew to serve **10,000+ customers**.

---

## Partnership Elements

| Component | Detail |
|-----------|--------|
| **Platform** | Amazon Bedrock |
| **Investment** | Up to $4B from Amazon |
| **Cloud** | AWS as primary provider |
| **Customers** | 10,000+ (as of partnership milestones) |

---

## Amazon Bedrock

**Amazon Bedrock** is AWS's fully managed service for foundation models. It provides:

- Access to Claude and other foundation models via API
- Serverless inference
- Fine-tuning and customization
- Enterprise security and compliance

---

## Strategic Collaboration

- **Claude as flagship** – Claude was positioned as a leading model on Bedrock
- **Trainium and Inferentia** – Use of Amazon's custom AI chips for training and inference
- **Enterprise distribution** – AWS's large enterprise customer base
- **Global availability** – Bedrock availability across AWS regions

---

## Customer Growth

The partnership reported **10,000+ customers** using Claude via Bedrock, including enterprises across industries such as finance, healthcare, and technology.

---

## Later Developments

- **November 2024:** Amazon committed an additional $4B (total $8B) – see [20241100-amazon-additional-4b.md](20241100-amazon-additional-4b.md)
- Claude continued to be a core offering on Bedrock
