# Claude for Enterprise 发布

**日期：** 2024年9月4日  
**类型：** 产品发布

## 描述

Claude for Enterprise 将 Anthropic 最先进的能力带给大型组织，具备企业级安全、合规和集成功能。该产品包括 500K 上下文、GitHub 集成以及身份管理的完整 SSO/SCIM 支持。

## 技术细节

- **上下文：** **500K tokens**——标准模型的 2.5 倍，支持大规模代码库和文档集分析
- **GitHub 集成：** 与 GitHub 直接集成，用于代码审查、PR 分析、仓库理解
- **身份与访问：**
  - **SSO：** 企业身份提供商的单点登录
  - **SCIM：** 跨域身份管理系统——自动化用户配置/取消配置
- **安全：** 企业级安全控制、数据处理、合规支持
- **管理员控制：** 集中管理、使用分析、策略执行
- **支持：** 专属企业支持

## 意义与影响

- **企业市场：** 满足大型组织（财富 500 强等）的需求
- **开发者工作流：** GitHub 集成针对软件开发团队
- **上下文领先：** 500K 上下文在代码库分析方面无与伦比
- **合规：** SSO/SCIM 是企业采购的必备条件
- **收入：** 企业层级代表最高 ARPU 细分市场

# Claude for Enterprise Launch

**Date:** September 4, 2024  
**Type:** Product Launch

## Description

Claude for Enterprise brought Anthropic's most advanced capabilities to large organizations with enterprise-grade security, compliance, and integration features. The offering included 500K context, GitHub integration, and full SSO/SCIM support for identity management.

## Technical Details

- **Context:** **500K tokens**—2.5x standard models, enabling analysis of massive codebases and document sets
- **GitHub Integration:** Direct integration with GitHub for code review, PR analysis, repository understanding
- **Identity & Access:**
  - **SSO:** Single Sign-On for enterprise identity providers
  - **SCIM:** System for Cross-domain Identity Management—automated user provisioning/deprovisioning
- **Security:** Enterprise-grade security controls, data handling, compliance support
- **Admin Controls:** Centralized management, usage analytics, policy enforcement
- **Support:** Dedicated enterprise support

## Significance & Impact

- **Enterprise Market:** Addressed requirements of large organizations (Fortune 500, etc.)
- **Developer Workflows:** GitHub integration targeted software development teams
- **Context Leadership:** 500K context unmatched for codebase analysis
- **Compliance:** SSO/SCIM required for enterprise procurement
- **Revenue:** Enterprise tier represented highest ARPU segment
