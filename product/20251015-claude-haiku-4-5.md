# Claude Haiku 4.5 发布

**日期：** 2025年10月15日  
**类型：** 产品发布  
**分类：** AI 模型发布

## 概述

Claude Haiku 4.5 以显著更低的成本和更快的速度提供接近前沿的性能。它在代理式编程能力上达到 Claude Sonnet 4.5 约 90% 的水平，同时运行速度快 4-5 倍。

## 关键技术细节

| 指标 | 数值 |
|--------|-------|
| **SWE-bench 性能** | Sonnet 4.5 的约 90% |
| **速度** | 比 Sonnet 4.5 快 4-5 倍 |
| **定价（输入）** | 每百万 tokens 1 美元 |
| **定价（输出）** | 每百万 tokens 5 美元 |
| **上下文窗口** | 200K tokens |
| **最大输出** | 64K tokens |
| **Extended Thinking** | 支持 |

## 能力

- 以一小部分成本实现接近前沿的性能
- 在某些 computer use 任务上超越 Claude Sonnet 4
- 强劲的代理式编程能力——以 4-5 倍速度达到 Sonnet 4.5 约 90% 的水平
- 支持复杂推理的扩展思考
- 适用于高吞吐量、成本敏感型应用

## 可用性

- Anthropic API
- Amazon Bedrock
- Google Vertex AI
- GitHub Copilot

## 意义

Haiku 4.5 代表了 Anthropic 持续致力于在不同性价比点提供一系列模型的承诺。通过以大幅更低的成本和更高的速度实现接近 Sonnet 级的能力，它使前沿级 AI 能够在成本敏感型应用和高容量用例中更广泛地部署。

# Claude Haiku 4.5 Launch

**Date:** October 15, 2025  
**Type:** Product Launch  
**Category:** AI Model Release

## Overview

Claude Haiku 4.5 delivers near-frontier performance at significantly lower cost and faster speed than its larger siblings. It achieves approximately 90% of Claude Sonnet 4.5's agentic coding capabilities while running 4-5x faster.

## Key Technical Details

| Metric | Value |
|--------|-------|
| **SWE-bench Performance** | ~90% of Sonnet 4.5 |
| **Speed** | 4-5x faster than Sonnet 4.5 |
| **Pricing (Input)** | $1 per million tokens |
| **Pricing (Output)** | $5 per million tokens |
| **Context Window** | 200K tokens |
| **Max Output** | 64K tokens |
| **Extended Thinking** | Supported |

## Capabilities

- Near-frontier performance at a fraction of the cost
- Surpasses Claude Sonnet 4 on some computer use tasks
- Strong agentic coding capabilities — approximately 90% of Sonnet 4.5 at 4-5x the speed
- Extended thinking support for complex reasoning
- Suitable for high-throughput, cost-sensitive applications

## Availability

- Anthropic API
- Amazon Bedrock
- Google Vertex AI
- GitHub Copilot

## Significance

Haiku 4.5 represents Anthropic's continued commitment to offering a range of models at different price-performance points. By achieving near-Sonnet-level capabilities at dramatically lower cost and higher speed, it enables broader deployment of frontier-class AI across cost-sensitive applications and high-volume use cases.
