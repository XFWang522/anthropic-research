# Build Artifacts 发布

**日期：** 2025年6月25日  
**类型：** 产品发布

## 描述

Build Artifacts 使用户能够无需编写代码即可创建交互式 AI 应用。基于 Artifacts 功能，用户可以通过与 Claude 的自然语言对话构建应用，然后与他人分享。该功能为非开发者实现了应用创建的民主化。

## 技术细节

- **无代码创建：** 在对话中描述期望的应用；Claude 生成交互式应用
- **交互式：** 应用可包含表单、按钮、逻辑、数据显示
- **可分享：** 构建的 artifacts 可通过链接分享供他人使用
- **基础：** 基于 Artifacts 基础设施（代码、HTML/CSS/JS 生成）
- **用例：** 调查、计算器、简单工具、数据仪表板、表单、游戏
- **可用性：** 集成到 claude.ai

## 意义与影响

- **民主化：** 非编码者可以创建功能型应用
- **产品扩展：** 将 Claude 从问答扩展到应用创建
- **参与度：** 新用例增加了用户互动和留存
- **病毒潜力：** 可分享的应用可以推动用户获取
- **差异化：** 与专注于纯聊天的竞争对手的独特能力

# Build Artifacts Launch

**Date:** June 25, 2025  
**Type:** Product Launch

## Description

Build Artifacts enabled users to create interactive AI applications without writing code. Building on the Artifacts feature, users could construct apps through natural language conversation with Claude, then share them with others. The feature democratized app creation for non-developers.

## Technical Details

- **No-Code Creation:** Describe desired app in conversation; Claude generates interactive application
- **Interactive:** Apps could include forms, buttons, logic, data display
- **Shareable:** Built artifacts could be shared via link for others to use
- **Foundation:** Built on Artifacts infrastructure (code, HTML/CSS/JS generation)
- **Use Cases:** Surveys, calculators, simple tools, data dashboards, forms, games
- **Availability:** Integrated into claude.ai

## Significance & Impact

- **Democratization:** Non-coders could create functional applications
- **Product Expansion:** Moved Claude beyond Q&A to app creation
- **Engagement:** New use case increased user interaction and retention
- **Viral Potential:** Shareable apps could drive user acquisition
- **Differentiation:** Unique capability vs. competitors focused on chat-only
