# Claude 3 系列发布：Opus、Sonnet、Haiku

**日期：** 2024年3月4日  
**类型：** 产品发布

## 描述

Claude 3 系列代表了 Anthropic 的首个多模型产品线，提供针对不同用例优化的三个层级：Opus（最强能力）、Sonnet（平衡）、Haiku（快速且经济）。所有模型均具备视觉能力、200K 上下文，以及拒绝行为的显著改进。

该发布将 Claude 的可用性扩展到 159 个国家，确立了 Anthropic 作为全球 AI 提供商的地位。

## 技术细节

- **模型层级：**
  - **Claude 3 Opus：** 最高智能，复杂任务，细致分析
  - **Claude 3 Sonnet：** 平衡的性能和速度，通用用途
  - **Claude 3 Haiku：** 最快、最经济，高容量应用

- **统一功能：**
  - 所有模型**200K 上下文**
  - **视觉：** 图像理解和分析（文档、图表、示意图）
  - 相比 Claude 2 **拒绝减少 2 倍**——更好的安全校准，避免过度拒绝

- **可用性：** 通过 API 和 claude.ai 覆盖 159 个国家

- **API：** 三个模型均提供分层定价

## 意义与影响

- **产品策略：** 分层模型系列实现了市场细分和成本优化
- **多模态：** 视觉能力将用例扩展到文档分析、图像理解
- **安全校准：** 减少不必要的拒绝改善了用户体验，同时保持安全
- **全球覆盖：** 159 个国家代表了重大地理扩张
- **企业信号：** Opus/Sonnet/Haiku 命名和定位为企业销售做好准备

# Claude 3 Family Launch: Opus, Sonnet, Haiku

**Date:** March 4, 2024  
**Type:** Product Launch

## Description

The Claude 3 family represented Anthropic's first multi-model product line, offering three tiers optimized for different use cases: Opus (most capable), Sonnet (balanced), and Haiku (fast and affordable). All models featured vision capabilities, 200K context, and significant improvements in refusal behavior.

The launch expanded Claude's availability to 159 countries, establishing Anthropic as a global AI provider.

## Technical Details

- **Model Tiers:**
  - **Claude 3 Opus:** Highest intelligence, complex tasks, nuanced analysis
  - **Claude 3 Sonnet:** Balanced performance and speed, general-purpose
  - **Claude 3 Haiku:** Fastest, most affordable, high-volume applications

- **Unified Features:**
  - **200K context** across all models
  - **Vision:** Image understanding and analysis (documents, charts, diagrams)
  - **2x fewer refusals** compared to Claude 2—better calibrated safety without over-refusal

- **Availability:** 159 countries via API and claude.ai

- **API:** All three models available with tiered pricing

## Significance & Impact

- **Product Strategy:** Tiered model family enabled market segmentation and cost optimization
- **Multimodal:** Vision capability expanded use cases to document analysis, image understanding
- **Safety Calibration:** Fewer unnecessary refusals improved user experience while maintaining safety
- **Global Reach:** 159 countries represented major geographic expansion
- **Enterprise Signal:** Opus/Sonnet/Haiku naming and positioning prepared for enterprise sales
