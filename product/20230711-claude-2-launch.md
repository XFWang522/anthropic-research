# Claude 2 发布

**日期：** 2023年7月11日  
**类型：** 产品发布

## 描述

Claude 2 相比 Claude 1 实现了重大能力升级，在编程、数学、推理和安全方面都有显著改进。该模型取得了强劲的基准测试结果，包括 Bar exam 76.5% 和 HumanEval 71.2%，同时安全改进翻倍。

Claude 2 通过 API 和新的 claude.ai 聊天界面提供，扩大了 Anthropic 对开发者和普通用户的覆盖。

## 技术细节

- **基准性能：**
  - **Bar Exam：** 76.5%（前 10% 考生水平）
  - **HumanEval：** 71.2%（编程）
  - **GSM8K：** 88%（小学数学）

- **上下文窗口：** 保持 Claude 1 的 100K tokens

- **安全：** 相比 Claude 1 安全指标提升 2 倍——更少有害输出、更好的拒绝行为

- **可用性：**
  - 面向开发者的 API
  - claude.ai 网页界面直接访问
  - 与 DuckDuckGo、Notion 等持续合作

- **用例：** 强调编程辅助、分析、创意写作和问答

## 意义与影响

- **能力飞跃：** 证明 Anthropic 可以在基准测试上与领先模型竞争
- **安全+能力：** 证明安全改进可以与能力提升同时实现
- **消费者访问：** claude.ai 将 Claude 带给 API 客户以外的普通用户
- **编程重点：** 强劲的 HumanEval 分数将 Claude 定位于开发者工具市场
- **信任建设：** 安全改进强化了 Anthropic 对齐优先的定位

# Claude 2 Launch

**Date:** July 11, 2023  
**Type:** Product Launch

## Description

Claude 2 represented a major capability upgrade over Claude 1, with significant improvements across coding, math, reasoning, and safety. The model achieved strong benchmark results including 76.5% on the Bar exam and 71.2% on HumanEval, while doubling safety improvements.

Claude 2 was made available via API and a new claude.ai chat interface, expanding Anthropic's reach to both developers and general users.

## Technical Details

- **Benchmark Performance:**
  - **Bar Exam:** 76.5% (top 10% of test-takers)
  - **HumanEval:** 71.2% (coding)
  - **GSM8K:** 88% (grade school math)

- **Context Window:** 100K tokens maintained from Claude 1

- **Safety:** 2x improvement in safety metrics compared to Claude 1—fewer harmful outputs, better refusal behavior

- **Availability:**
  - API for developers
  - claude.ai web interface for direct access
  - Continued partnerships with DuckDuckGo, Notion, and others

- **Use Cases:** Emphasized coding assistance, analysis, creative writing, and Q&A

## Significance & Impact

- **Capability Leap:** Demonstrated Anthropic could compete on benchmarks with leading models
- **Safety + Capability:** Proved safety improvements could coincide with capability gains
- **Consumer Access:** claude.ai brought Claude to general users beyond API customers
- **Coding Focus:** Strong HumanEval score positioned Claude for developer tools market
- **Trust Building:** Safety improvements reinforced Anthropic's alignment-first positioning
