# Computer Use 能力 - 公开 Beta

**日期：** 2024年10月22日  
**类型：** 产品发布 / 研究

## 描述

Computer Use 代表了 Anthropic 的首个"前沿 AI"能力——使 Claude 能够通过截图、光标控制和键盘输入与计算机界面交互。该模型在 OSWorld 基准上达到 14.9%，展示了完成实际计算机任务的能力。

这标志着 AI 代理能够在任意软件环境中操作的重要一步。

## 技术细节

- **能力：**
  - **截图：** 接收和解释屏幕状态
  - **光标：** 移动和点击鼠标
  - **输入：** 键盘输入
  - **多步骤任务：** 导航应用、填写表单、完成工作流

- **OSWorld 基准：** 14.9%——评估在模拟环境中完成多样化计算机任务的能力
- **架构：** 视觉模型 + 动作生成；接收像素输入，输出控制动作
- **安全：** 沙盒环境；谨慎的提示设计以防止有害操作
- **可用性：** 面向选定用户的公开 beta

## 意义与影响

- **代理式 AI：** 超越文本，实现实际计算机交互
- **首个前沿：** Anthropic 首个具备 computer-use 能力的模型
- **用例：** 自动化、无障碍、跨任意软件的复杂工作流
- **研究方向：** 为未来代理能力奠定基础
- **基准：** OSWorld 建立了 computer-use 评估的基线

# Computer Use Capability - Public Beta

**Date:** October 22, 2024  
**Type:** Product Launch / Research

## Description

Computer Use represented Anthropic's first "frontier AI" capability—enabling Claude to interact with computer interfaces through screenshots, cursor control, and keyboard input. The model achieved 14.9% on the OSWorld benchmark, demonstrating ability to complete real-world computer tasks.

This marked a significant step toward AI agents that can operate in arbitrary software environments.

## Technical Details

- **Capabilities:**
  - **Screenshots:** Receive and interpret screen state
  - **Cursor:** Move and click mouse
  - **Type:** Keyboard input
  - **Multi-step tasks:** Navigate applications, fill forms, complete workflows

- **OSWorld Benchmark:** 14.9%—evaluates ability to complete diverse computer tasks in simulated environments
- **Architecture:** Vision model + action generation; receives pixel input, outputs control actions
- **Safety:** Sandboxed environment; careful prompt design to prevent harmful actions
- **Availability:** Public beta for select users

## Significance & Impact

- **Agentic AI:** Moved beyond text to real-world computer interaction
- **First Frontier:** Anthropic's first computer-use capable model
- **Use Cases:** Automation, accessibility, complex workflows across any software
- **Research Direction:** Foundation for future agent capabilities
- **Benchmark:** OSWorld established baseline for computer-use evaluation
