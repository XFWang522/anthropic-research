# Claude Sonnet 4.6 发布

**日期：** 2026年2月17日  
**类型：** 产品发布

## 描述

Claude Sonnet 4.6 将 100 万 token 上下文带到 Sonnet 层级，并获得了 ASL-3（Anthropic Safety Level 3）认证。定价为每百万 tokens 3 美元/15 美元，该模型成为 claude.ai 的默认模型——代表 Anthropic 全面可用的旗舰产品。

## 技术细节

- **上下文：** **100 万 tokens**——与 Opus 4.6 相同，让长上下文普及
- **ASL-3：** Anthropic Safety Level 3 认证——验证的安全标准
- **定价：** 每百万输入 3 美元，每百万输出 15 美元（从 Sonnet 4.5 保持）
- **默认模型：** **claude.ai 默认**——新用户的首选体验
- **定位：** 广泛使用的平衡能力、安全和成本
- **API：** 完整可用

## 意义与影响

- **100 万普惠：** 长上下文不再局限于高级 Opus 层级
- **默认选择：** 作为 claude.ai 默认，Sonnet 4.6 定义了 Anthropic 的主流产品
- **ASL-3：** 安全认证强化了 Anthropic 的信任定位
- **价值：** 3 美元/15 美元的 100 万上下文代表了卓越价值
- **采用：** 默认地位推动了最大曝光和使用

# Claude Sonnet 4.6 Launch

**Date:** February 17, 2026  
**Type:** Product Launch

## Description

Claude Sonnet 4.6 brought 1 million token context to the Sonnet tier and achieved ASL-3 (Anthropic Safety Level 3) certification. Priced at $3/$15 per million tokens, the model became the default on claude.ai—representing Anthropic's flagship for general availability.

## Technical Details

- **Context:** **1M tokens**—same as Opus 4.6, democratizing long-context access
- **ASL-3:** Anthropic Safety Level 3 certification—validated safety standards
- **Pricing:** $3 per million input, $15 per million output (maintained from Sonnet 4.5)
- **Default Model:** **Default on claude.ai**—first experience for new users
- **Positioning:** Balanced capability, safety, and cost for broad use
- **API:** Full availability

## Significance & Impact

- **1M for All:** Long context no longer limited to premium Opus tier
- **Default Choice:** As claude.ai default, Sonnet 4.6 defined Anthropic's mainstream offering
- **ASL-3:** Safety certification reinforced Anthropic's trust positioning
- **Value:** 1M context at $3/$15 represented exceptional value
- **Adoption:** Default status drove maximum exposure and usage
