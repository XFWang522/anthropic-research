# Claude 4（Opus 4 与 Sonnet 4）发布 - 开发者大会

**日期：** 2025年5月22日  
**类型：** 产品发布

## 描述

Claude 4 在 Anthropic 开发者大会上发布，推出 Opus 4 和 Sonnet 4，带来重大能力升级。Opus 4 在 SWE-bench 上达到 72.5%，可处理跨越 7+ 小时的任务。Sonnet 4 保持 3 美元/15 美元定价，支持并行工具使用。该发布标志着 Anthropic 的下一代模型系列。

## 技术细节

- **Claude Opus 4：**
  - **SWE-bench：** 72.5%（软件工程基准）
  - **长任务：** 7+ 小时任务能力
  - 扩展推理
  - 各基准最佳水平

- **Claude Sonnet 4：**
  - **定价：** 每百万 tokens 3 美元/15 美元（保持）
  - **并行工具使用：** 同时调用多个工具——更快的代理式工作流
  - 强劲的编程和通用能力

- **开发者大会：** 以开发者为重点的重大发布活动
- **API：** 两个模型均提供完整 API

## 意义与影响

- **编程领先：** 72.5% SWE-bench 为编程模型设定了新基准
- **长任务：** 7+ 小时能力实现了复杂、多步骤工作流
- **并行工具：** 代理式应用的重大 UX 改进
- **定价：** Sonnet 4 尽管能力提升仍保持积极定价
- **开发者重点：** 大会强调了 Anthropic 的开发者生态

# Claude 4 (Opus 4 & Sonnet 4) Launch - Developer Conference

**Date:** May 22, 2025  
**Type:** Product Launch

## Description

Claude 4 launched at Anthropic's developer conference, introducing Opus 4 and Sonnet 4 with major capability upgrades. Opus 4 achieved 72.5% on SWE-bench and could handle tasks spanning 7+ hours. Sonnet 4 maintained $3/$15 pricing with parallel tool use. The launch marked Anthropic's next-generation model family.

## Technical Details

- **Claude Opus 4:**
  - **SWE-bench:** 72.5% (software engineering benchmark)
  - **Long tasks:** 7+ hour task capability
  - Extended reasoning
  - Best-in-class across benchmarks

- **Claude Sonnet 4:**
  - **Pricing:** $3/$15 per million tokens (maintained)
  - **Parallel tool use:** Call multiple tools simultaneously—faster agentic workflows
  - Strong coding and general capabilities

- **Developer Conference:** Major launch event with developer-focused announcements
- **API:** Full API availability for both models

## Significance & Impact

- **Coding Leadership:** 72.5% SWE-bench set new benchmark for coding models
- **Long Tasks:** 7+ hour capability enabled complex, multi-step workflows
- **Parallel Tools:** Significant UX improvement for agentic applications
- **Pricing:** Sonnet 4 maintained aggressive pricing despite capability gains
- **Developer Focus:** Conference emphasized Anthropic's developer ecosystem
