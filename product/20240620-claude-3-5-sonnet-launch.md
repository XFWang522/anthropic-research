# Claude 3.5 Sonnet 发布

**日期：** 2024年6月20日  
**类型：** 产品发布

## 描述

Claude 3.5 Sonnet 带来了突破：以 Sonnet 定价提供 Opus 级智能。该模型在编程方面表现出色——在代理式编程基准上达到 64%——并推出了 Artifacts，一项预览功能，允许 Claude 在对话旁边创建和显示交互式内容。

每百万 tokens 3 美元/15 美元的定价使其在开发者用例上极具竞争力。

## 技术细节

- **能力：** 以 Sonnet 层级定价提供 Opus 级性能
- **编程：** **64% 代理式编程**——代码生成和编辑的最佳水平
- **定价：** 每百万输入 tokens 3 美元，每百万输出 tokens 15 美元
- **Artifacts（预览）：** Claude 可以生成并渲染交互式内容——代码、文档、图表——在聊天旁边的专用面板中
- **上下文：** 200K tokens
- **视觉：** 完整视觉能力
- **API：** 立即向所有客户提供

## 意义与影响

- **价值主张：** Opus 智能 + Sonnet 成本颠覆了定价预期
- **开发者重点：** 64% 代理式编程将 Claude 定位为最佳编程模型
- **Artifacts：** 交互式 AI 输出的预览——为 2025 年 Build Artifacts 奠定基础
- **市场份额：** 以编程 + 定价组合从竞争对手吸引开发者
- **基准：** 为编程模型性能设定了新标准

# Claude 3.5 Sonnet Launch

**Date:** June 20, 2024  
**Type:** Product Launch

## Description

Claude 3.5 Sonnet delivered a breakthrough: Opus-level intelligence at Sonnet pricing. The model excelled at coding—achieving 64% on agentic coding benchmarks—and introduced Artifacts, a preview feature allowing Claude to create and display interactive content alongside conversations.

Pricing at $3/$15 per million tokens made it highly competitive for developer use cases.

## Technical Details

- **Capability:** Opus-level performance at Sonnet-tier pricing
- **Coding:** **64% agentic coding**—best-in-class for code generation and editing
- **Pricing:** $3 per million input tokens, $15 per million output tokens
- **Artifacts (Preview):** Claude could generate and render interactive content—code, documents, diagrams—in a dedicated panel alongside the chat
- **Context:** 200K tokens
- **Vision:** Full vision capability
- **API:** Available immediately for all customers

## Significance & Impact

- **Value Proposition:** Opus intelligence at Sonnet cost disrupted pricing expectations
- **Developer Focus:** 64% agentic coding positioned Claude as best coding model
- **Artifacts:** Preview of interactive AI outputs—foundation for Build Artifacts (2025)
- **Market Share:** Attracted developers from competitors with coding + pricing combination
- **Benchmark:** Set new bar for coding model performance
