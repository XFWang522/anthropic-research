# Agent Capabilities API Beta 发布

**日期：** 2025年5月22日  
**类型：** 产品发布

## 描述

Agent Capabilities API beta 为开发者提供了对高级代理功能的统一访问：Code Execution、MCP Connector、Files API 和 Extended Prompt Caching。这些能力使开发者能够构建可以执行代码、连接工具、管理文件并高效缓存长对话的复杂 AI 代理。

## 技术细节

- **Code Execution：** 模型运行代码、查看结果、迭代的沙盒环境
- **MCP Connector：** 内置 Model Context Protocol 支持——连接工具、数据源、提示
- **Files API：** 在代理上下文中上传、管理和引用文件
- **Extended Prompt Caching：** 缓存长系统提示或上下文，实现跨请求的成本/延迟效率
- **统一 API：** 所有代理能力的单一接口
- **可用性：** 面向开发者的 Beta 访问

## 意义与影响

- **代理开发：** 降低了构建生产 AI 代理的门槛
- **MCP 集成：** 原生 MCP 使工具集成无缝
- **Code Execution：** 对编程代理、数据分析、自动化至关重要
- **成本效率：** 提示缓存降低了长上下文应用的成本
- **生态：** 与 Claude 4 发布一起强化了 Anthropic 的开发者平台

# Agent Capabilities API Beta

**Date:** May 22, 2025  
**Type:** Product Launch

## Description

The Agent Capabilities API beta provided developers with unified access to advanced agent features: Code Execution, MCP Connector, Files API, and Extended Prompt Caching. These capabilities enabled building sophisticated AI agents that could execute code, connect to tools, manage files, and efficiently cache long conversations.

## Technical Details

- **Code Execution:** Sandboxed environment for model to run code, see results, iterate
- **MCP Connector:** Built-in Model Context Protocol support—connect to tools, data sources, prompts
- **Files API:** Upload, manage, and reference files in agent context
- **Extended Prompt Caching:** Cache long system prompts or context for cost/latency efficiency across requests
- **Unified API:** Single interface for all agent capabilities
- **Availability:** Beta access for developers

## Significance & Impact

- **Agent Development:** Lowered barrier to building production AI agents
- **MCP Integration:** Native MCP made tool integration seamless
- **Code Execution:** Critical for coding agents, data analysis, automation
- **Cost Efficiency:** Prompt caching reduced costs for long-context applications
- **Ecosystem:** Strengthened Anthropic's developer platform alongside Claude 4 launch
