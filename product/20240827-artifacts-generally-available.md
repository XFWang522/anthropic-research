# Artifacts 全面可用

**日期：** 2024年8月27日  
**类型：** 产品发布

## 描述

Artifacts 从预览升级为面向所有 Claude 用户的全面可用（GA）。该功能允许 Claude 在对话旁边的专用侧边栏中创建、编辑和显示交互式内容——代码、文档、图表、网页。预览期间已创建数千万个 Artifacts，验证了强烈的用户需求。

## 技术细节

- **Artifacts：** Claude 渲染生成内容的专用面板
- **内容类型：**
  - 代码（带语法高亮）
  - 文档（Markdown、格式化文本）
  - 图表和可视化
  - 网页（HTML/CSS/JS）
  - 交互式应用

- **工作流：** 用户请求内容 → Claude 生成 → 在 Artifact 面板中渲染 → 用户可以编辑、迭代或导出
- **可用性：** claude.ai 上的所有用户（免费和付费）
- **规模：** 预览期间创建了数千万个 Artifacts

## 意义与影响

- **产品市场契合：** 数千万创建量证明了明确的用户价值
- **差异化：** 与竞争对手独特的 UX——在对话旁边进行内容创建
- **开发者采用：** 代码 Artifacts 加速了开发者工作流
- **基础：** 为 2025 年 Build Artifacts（无需编码的交互式应用）铺平道路
- **参与度：** 增加了会话深度和产品内停留时间

# Artifacts Generally Available

**Date:** August 27, 2024  
**Type:** Product Launch

## Description

Artifacts graduated from preview to generally available (GA) for all Claude users. The feature allowed Claude to create, edit, and display interactive content—code, documents, diagrams, web pages—in a dedicated side panel alongside the conversation. Tens of millions of Artifacts had been created during the preview period, validating strong user demand.

## Technical Details

- **Artifacts:** Dedicated panel where Claude renders generated content
- **Content Types:**
  - Code (with syntax highlighting)
  - Documents (Markdown, formatted text)
  - Diagrams and visualizations
  - Web pages (HTML/CSS/JS)
  - Interactive applications

- **Workflow:** User requests content → Claude generates → Renders in Artifact panel → User can edit, iterate, or export
- **Availability:** All users on claude.ai (free and paid)
- **Scale:** Tens of millions of Artifacts created during preview

## Significance & Impact

- **Product-Market Fit:** Tens of millions created demonstrated clear user value
- **Differentiation:** Unique UX vs. competitors—content creation alongside conversation
- **Developer Adoption:** Code Artifacts accelerated developer workflows
- **Foundation:** Paved way for Build Artifacts (interactive apps without coding) in 2025
- **Engagement:** Increased session depth and time-in-product
