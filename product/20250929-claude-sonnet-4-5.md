# Claude Sonnet 4.5 发布

**日期：** 2025年9月29日  
**类型：** 产品发布

## 描述

Claude Sonnet 4.5 以 77.2% 创下 SWE-bench 历史最高分，具备对复杂任务 30+ 小时的专注能力，以及在 Replit 基准上 0% 错误率。该模型代表了 Sonnet 层级编程能力的顶峰，以 Sonnet 定价提供 Opus 级编程。

## 技术细节

- **SWE-bench：** **77.2%**——所有模型历史最高
- **专注：** 30+ 小时任务能力——对复杂、多步骤项目的扩展专注
- **Replit：** **0% 错误率**——在 Replit 编程基准上的完美表现
- **定价：** 保持 Sonnet 层级定价
- **用例：** 完整软件开发生命周期、复杂重构、长时间运行的编程任务
- **API：** 面向所有客户可用

## 意义与影响

- **编程顶峰：** 77.2% SWE-bench 创下行业新纪录
- **价值：** 以 Sonnet 定价的最佳编程模型——卓越的价值主张
- **Replit：** 0% 错误展示了生产编程的可靠性
- **长任务：** 30+ 小时能力实现了严肃的软件项目
- **市场定位：** 巩固了 Claude 作为权威编程助手的地位

Claude Sonnet 4.5 achieved the highest SWE-bench score ever at 77.2%, with capability for 30+ hour focus on complex tasks and 0% error rate on the Replit benchmark. The model represented the pinnacle of coding capability in the Sonnet tier, offering Opus-level coding at Sonnet pricing.

- **Focus:** 30+ hour task capability—extended focus on complex, multi-step projects
- **Replit:** **0% error rate**—perfect performance on Replit coding benchmark
- **Pricing:** Maintained Sonnet-tier pricing
- **Use Cases:** Full software development lifecycle, complex refactoring, long-running coding tasks
- **API:** Available for all customers

- **Coding Pinnacle:** 77.2% SWE-bench set new industry record

- **Replit:** 0% error demonstrated reliability for production coding
- **Long Tasks:** 30+ hour capability enabled serious software projects
