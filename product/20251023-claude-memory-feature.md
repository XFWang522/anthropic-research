# Claude Memory 功能发布

**日期：** 2025年10月23日  
**类型：** 产品发布

## 描述

Claude Memory 引入了持久的、基于项目的记忆，使 Claude 能够跨对话记住用户偏好、上下文和信息。用户可以从 ChatGPT 和 Gemini 导入记忆，隐身模式确保在不需要记忆时的隐私。

## 技术细节

- **基于项目：** 记忆按项目组织——不同用例的不同上下文
- **持久性：** Claude 在项目内的会话间记住
- **导入：** 从 ChatGPT 和 Gemini 导入记忆——从竞争对手切换的用户的迁移路径
- **隐身模式：** 无记忆的对话——敏感话题的隐私
- **用户控制：** 用户可以查看、编辑和删除记忆
- **用例：** 个性化、项目上下文、偏好（写作风格、编码规范等）

## 意义与影响

- **个性化：** 记忆实现了真正个性化的 AI 辅助
- **留存：** 项目连续性增加了用户参与度和粘性
- **迁移：** ChatGPT/Gemini 导入降低了切换成本
- **隐私：** 隐身模式解决了隐私关切
- **竞争：** 以导入差异化匹配竞争对手的记忆功能

# Claude Memory Feature Launch

**Date:** October 23, 2025  
**Type:** Product Launch

## Description

Claude Memory introduced persistent, project-based memory that allowed Claude to remember user preferences, context, and information across conversations. Users could import memories from ChatGPT and Gemini, and an incognito mode ensured privacy when memory was not desired.

## Technical Details

- **Project-Based:** Memories organized by project—different contexts for different use cases
- **Persistence:** Claude remembers across sessions within a project
- **Import:** Import memories from ChatGPT and Gemini—migration path for users switching from competitors
- **Incognito Mode:** Conversations without memory—privacy for sensitive topics
- **User Control:** Users could view, edit, and delete memories
- **Use Cases:** Personalization, project context, preferences (writing style, coding conventions, etc.)

## Significance & Impact

- **Personalization:** Memory enabled truly personalized AI assistance
- **Retention:** Project continuity increased user engagement and stickiness
- **Migration:** ChatGPT/Gemini import reduced switching costs
- **Privacy:** Incognito mode addressed privacy concerns
- **Competitive:** Matched competitor memory features with import differentiation
