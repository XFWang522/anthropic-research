# Claude 3.7 Sonnet 与 Claude Code 发布

**日期：** 2025年2月24日  
**类型：** 产品发布

## 描述

Claude 3.7 Sonnet 推出了首个"混合推理"模型，具备扩展思考能力——允许 Claude 在回应前通过更长的内部 deliberation 推理复杂问题。该发布还包括 Claude Code，面向开发者的专用 CLI 工具，以及用于处理多步骤推理任务的 Extended Thinking 功能。

## 技术细节

- **混合推理：** 首个将标准生成与扩展"思考"阶段相结合的模型
- **Extended Thinking：** 模型可以在产生最终答案前花费更多计算进行推理——根据配置对用户可见或不可见
- **Claude Code：** 面向开发者的命令行界面
  - 终端集成
  - 代码生成和编辑
  - 项目感知辅助
- **用例：** 复杂数学、编程、多步骤规划、需要深度推理的分析
- **定价：** Sonnet 层级定价，扩展思考为可选能力

## 意义与影响

- **推理创新：** 扩展思考代表了复杂推理的新架构方法
- **开发者工具：** Claude Code CLI 直接满足开发者工作流
- **基础：** 为 Claude 4 的推理能力和 Tracing the Thoughts 研究铺平道路
- **差异化：** 混合推理使 Claude 区别于仅 chain-of-thought 的方法
- **编程重点：** 强化了 Claude 作为领先编程助手的地位

# Claude 3.7 Sonnet & Claude Code Launch

**Date:** February 24, 2025  
**Type:** Product Launch

## Description

Claude 3.7 Sonnet introduced the first "hybrid reasoning" model with extended thinking capabilities—allowing Claude to reason through complex problems with longer internal deliberation before responding. The launch also included Claude Code, a dedicated CLI tool for developers, and the Extended Thinking feature for tackling multi-step reasoning tasks.

## Technical Details

- **Hybrid Reasoning:** First model combining standard generation with extended "thinking" phase
- **Extended Thinking:** Model can spend more compute on reasoning before producing final answer—visible or invisible to user depending on configuration
- **Claude Code:** Command-line interface for developers
  - Terminal integration
  - Code generation and editing
  - Project-aware assistance
- **Use Cases:** Complex math, coding, multi-step planning, analysis requiring deep reasoning
- **Pricing:** Sonnet-tier pricing with extended thinking as optional capability

## Significance & Impact

- **Reasoning Innovation:** Extended thinking represented new architectural approach to complex reasoning
- **Developer Tooling:** Claude Code CLI addressed developer workflow directly
- **Foundation:** Paved way for Claude 4's reasoning capabilities and Tracing the Thoughts research
- **Differentiation:** Hybrid reasoning distinguished Claude from chain-of-thought-only approaches
- **Coding Focus:** Reinforced Claude's position as leading coding assistant
