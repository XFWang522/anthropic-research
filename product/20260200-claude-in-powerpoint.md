# Claude in PowerPoint - 研究预览

**日期：** 2026年2月  
**类型：** 产品发布 / 合作

## 描述

Claude in PowerPoint 以研究预览形式将 AI 能力带入微软的演示软件。Microsoft 365 插件使用户能够使用自然语言与 Claude 生成和编辑幻灯片，将 AI 直接集成到演示工作流中。

该合作代表了 Anthropic 向生产力软件的扩张以及微软更广泛的 AI 生态。

## 技术细节

- **平台：** PowerPoint 的 Microsoft 365 插件
- **能力：**
  - **生成幻灯片：** 从大纲、主题或描述创建演示
  - **编辑幻灯片：** 使用自然语言指令修改现有幻灯片
  - **内容创建：** 生成要点、演讲者备注、视觉元素
- **集成：** Claude API 集成到 PowerPoint 工作流
- **可用性：** 研究预览——有限推出
- **要求：** Microsoft 365 订阅

## 意义与影响

- **微软合作：** Claude 的主要分发渠道
- **生产力市场：** 进入演示软件——庞大的用户群
- **企业：** Microsoft 365 集成意味着企业分发
- **用例扩展：** 从聊天扩展到文档/演示创建
- **竞争：** 与 Copilot 和其他 AI 生产力集成竞争

# Claude in PowerPoint - Research Preview

**Date:** February 2026  
**Type:** Product Launch / Partnership

## Description

Claude in PowerPoint brought AI capabilities to Microsoft's presentation software as a research preview. The Microsoft 365 add-in allowed users to generate and edit slides using natural language with Claude, integrating AI directly into the presentation workflow.

The partnership represented Anthropic's expansion into productivity software and Microsoft's broader AI ecosystem.

## Technical Details

- **Platform:** Microsoft 365 add-in for PowerPoint
- **Capabilities:**
  - **Generate slides:** Create presentations from outlines, topics, or descriptions
  - **Edit slides:** Modify existing slides with natural language instructions
  - **Content creation:** Generate bullet points, speaker notes, visuals
- **Integration:** Claude API integrated into PowerPoint workflow
- **Availability:** Research preview—limited rollout
- **Requirements:** Microsoft 365 subscription

## Significance & Impact

- **Microsoft Partnership:** Major distribution channel for Claude
- **Productivity Market:** Entered presentation software—huge user base
- **Enterprise:** Microsoft 365 integration meant enterprise distribution
- **Use Case Expansion:** Beyond chat to document/presentation creation
- **Competitive:** Competed with Copilot and other AI productivity integrations
